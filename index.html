Projects

Automatic Window Closing using Arduino

"RainGuard" is an innovative project designed to automate window closure during rain using Arduino technology. The system employs a rain sensor, an Arduino board, a motor, and a relay module to create a smart window control mechanism. The primary goal is to enhance convenience and protection of living spaces by autonomously responding to rainfall.

Home Automation using 8051

"SmartHome 8051" is an intelligent home automation system designed to enhance comfort, security, and energy efficiency. The project employs the 8051 microcontroller, interfaced with various sensors and actuators, to control and monitor household devices.
